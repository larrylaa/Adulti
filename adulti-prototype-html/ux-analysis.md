# User Experience Analysis

## Primary User Needs
- Understand current money position quickly without complex setup.
- Receive clear, prioritized next steps instead of generic finance tips.
- Learn financial concepts in simple language exactly when needed.
- Track progress to stay motivated and reduce decision fatigue.

## Core Interaction Logic
- Home first: show a lightweight snapshot (checking, savings, debt, investments).
- Roadmap second: convert current state into a ranked action sequence.
- Learn in context: each roadmap step offers quick guidance and deep guide access.
- Track completion: users mark progress and see where they are in a high-level flow.
- Edit anytime: users can update balances and debt entries from quick-edit surfaces.
